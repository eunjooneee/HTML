Preformat(pre) 태그

        Preformat(pre) 태그는 공백을 한 개만 표시한다.

        그러므로 문자열 사이에 여러 공백이 있더라도 하나만 유효하다.

        blockquote처럼 들여쓰기 된다. 

        프로그램 소스와 같은 코드들을 표현할 때 유용하다.
    
        
Java Hello

        class Hello {
            public static void main(String[] arg) {
                System.out.println("Hello, Java!!!");
            }
        }
