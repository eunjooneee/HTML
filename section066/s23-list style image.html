목록 스타일 - 순서 없는 목록에 이미지 첨가

순서없는 목록에 불릿 대신 이미지를 넣을 수 있다.

list-style-image : none url("(bullet-image.jpg)

  • none : 기본값, list-style-image에서 지정한 속성으로
  • 이미지 : url(bullet-image.jpg)

list-style-image : image

  • disc : 기본값, 채운 원
  • circle : 비어있는 원
  • square : 채운 사각형
  • none : 불릿 없애기
