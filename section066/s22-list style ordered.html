목록 스타일 - 순서 있는 목록 스타일 변경

순서없는 목록의 숫자를 변경할 수 있다.

list-style-type

  1. decimal : 기본값, 1로 시작하는 십진수
  2. decimal-leading-zero : 앞에 0이 붙는 십진수(01, 02, 03, ...)
  3. lower-roman : 소문자 로마 숫자
  4. upper-roman : 대문자 로마 숫자
  5. lower-alpha : 소문자 알파벳
  6. upper-alpha : 대문자 알파벳

list-style-type : upper-roman

  1. disc : 기본값, 채운 원
  2. circle : 비어있는 원
  3. square : 채운 사각형
  4. none : 불릿 없애기

list-style-type : lower roman

  1. disc : 기본값, 채운 원

  4. none : 불릿 없애기
