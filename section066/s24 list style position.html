목록 스타일 - 목록에 들여쓰기 효과

목록에서 불릿이나 번호는 내용과 바깥쪽의 표시가 된다. 실제 내용이 시작되는 위치에 불릿이나 번호를 표시할 수 있다.

list-style-position : outside | inside

  • none : 기본값, list-style-image에서 지정한 속성으로
  • 이미지 : url(bullet-image.jpg)

list-style-position : outside

  • disc : 기본값, 채운 원
  • circle : 비어있는 원
  • square : 채운 사각형
  • none : 불릿 없애기

list-style-position : inside

  • disc : 기본값, 채운 원
  • circle : 비어있는 원
  • square : 채운 사각형
  • none : 불릿 없애기
